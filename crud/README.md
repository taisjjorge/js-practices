# Doguito 🐶

### Start

Instalando dependências do projeto: 
- **npm install**

Simulando API: 
- **npx json-server --watch db.json**

*Utilizar extensão Go Live ou alternativa(browser-sync) para rodar o projeto e evitar erros de CORS.

## Sobre o projeto

> Doguito é um projeto de CRUD simples, utilizando javascript assíncrono, com simulador de API json-server para representar dados mockados.
>


Techs utilizadas -> `HTML/CSS/JS`.
